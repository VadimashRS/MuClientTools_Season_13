To open a Mu data file,
	Drap&Drop your file to *.exe or use console command to execute with the filepath.
To convert a txt file / folder
	Drap&Drop it to *.exe or use console command to execute with the filepath.

MuOnline Files Version:
	Season XIII Ep1 (default, all test files are from this client version)
	Season XII & below + Season XIII Ep2 & above 
	 (may work, should check crypto keys / zip-password)

7z.dll is required to zip/unzip the Lang.mpr file.
